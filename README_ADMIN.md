# Админ панель для Telegram бота

Веб-панель для управления ботом с Flask.

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

Создайте файл `.env` или установите переменные окружения:

```env
# Админ панель
ADMIN_USERNAME=admin
ADMIN_PASSWORD=your_secure_password
ADMIN_SECRET_KEY=your_secret_key_for_sessions
```

## Запуск

### Запуск бота
```bash
python bot.py
```

### Запуск админ панели
```bash
python admin_panel.py
```

Панель будет доступна по адресу: http://localhost:5000

## Возможности панели

- 📊 Статистика: всего пользователей, активных за 24ч, отправок, забаненных
- 👥 Все пользователи: список с информацией и статусами
- ⚡ Активные пользователи: фильтр по времени активности (1ч, 6ч, 24ч, 7 дней)
- 📋 Логи: просмотр логов с фильтрацией (все, ошибки, бот, инфо)
- 📸 Медиа: все отправленные медиа файлы с информацией

## База данных

База данных SQLite создается автоматически при первом запуске (`bot_data.db`).

Таблицы:
- `users` - пользователи бота
- `logs` - логи событий
- `media_submissions` - отправленные медиа файлы
