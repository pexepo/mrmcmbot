# Telegram-бот для предложки

Бот написан на `Python + aiogram 3`.

Изображения экранов бот теперь ищет так:
`start.png` или `start.jpg`,
`send_work.png` или `send_work.jpg`.

Для экрана правил используется один обязательный файл:
`rules.png`.

Для экрана успешной отправки можно добавить отдельный файл:
`success.png`.
Если его пока нет, временно используется `send_work.png`.

Для отдельного Discord-баннера можно добавить:
`discord.png`.
Если его нет, используется `start.png`.

Файлы `art_rules.*` и `edit_rules.*` для запуска больше не нужны.

Что уже реализовано:

- стартовый приветственный пост с изображением;
- главное меню с нижней клавиатурой;
- отдельные экраны с кнопками под сообщением для:
  - правил
  - Discord
  - связи с создателем
  - успешной и неуспешной отправки
- кнопки главного меню:
  - `Отправить эдит`
  - `Отправить арт`
  - `Discord сервер`
  - `Связь с создателем`
- правила для эдита и арта;
- кнопки `Согласиться` и `Отказаться`;
- возврат назад в меню;
- приём работы одним сообщением;
- подтверждение после отправки;
- лог-канал с информацией о пользователе и копией работы;
- сохранение известных `@username` и имён в `users.json`;
- бан, разбан и просмотр бан-листа;
- антиспам на повторные отправки;
- логирование ошибок бота в лог-канал.

## Запуск

Установить зависимости:

```powershell
pip install -r requirements.txt
```

Запустить бота:

```powershell
python bot.py
```

## Команды

- `/start` или `/menu` - открыть меню
- `/ban user_id причина` - забанить пользователя
- `/unban user_id` - разбанить пользователя
- `/banlist` - посмотреть бан-лист

## Настройки

Главные настройки находятся в начале файла [bot.py](C:/Users/pexep/Downloads/booot/bot.py):

- `TOKEN`
- `ADMIN_IDS`
- `LOG_CHANNEL_ID`
- `DISCORD_URL`
- `CREATOR_URL`
- `SUBMISSION_COOLDOWN_SECONDS`

При желании их можно переопределить через переменные окружения с теми же именами.
